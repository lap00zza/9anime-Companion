1.0.0

  • Changed the color and effect of the switches in the dashboard. Thanks @WEBhicham.
  • Added Keyboard Shortcuts to settings. Find it under General settings.
  • Added support for downloading from RapidVideo.
  • All the modals now have a "X" button.
  • Changed scroll to player to scroll to top. Added a smooth scroll animation to it and made it global. The button and shortcut can now both be used throughout 9anime.
  • Changed native browser notifications to in site notifications.

1.0.0-beta.2

  • Added auto-update for the MyAnimeList integration. This can be found in the MyAnimeList configure box.
  • Fixed an issue which was causing the alternate search overlay to pop up when typing password and in the report episode box.

1.0.0-beta.1
  • Redesigned the popup. Thanks @densityx.
  • Improved the downloader. Now 9anime Companion can download from all the 9anime Servers (F2, F4 etc).
  • Redesigned the download modal. Thanks @yumiris.
  • Vastly improved the ad/popup blocker. The new ad remover uses a combination of web request blocking and page element removal/function overrides. In my tests it removed 100% ads/popups without the need for an external adblocker.
  • Added support for keyboard shortcuts.
  • Added alternate search overlay . Shortcut s.
  • Added scroll to player. Shortcut t.
  • Added recently watched anime. This can be accessed via the popup.
  • Vastly improved the core functionality of MyAnimeList integration. It can easily be expanded in the future to add features like Auto MAL sync, full featured add/edit etc
